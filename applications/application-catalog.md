# Application Catalog

This document lists potential applications that can be built on top of the E2R Core Specification.

The list is intentionally broad and non-normative.
Its purpose is to preserve ideas for future development.

---

- Software release history

# Future Applications

The E2R Core Specification intentionally does not limit future application domains.

Any application that models identifiable entities, identifiable events, and relationships between them may be implemented using E2R.
